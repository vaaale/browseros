# Settings → Appearance

Customize how the desktop looks.

---

## Fields

- **Wallpaper** — accepts any of:
  - a **preset** id (a built‑in gradient),
  - an **image URL**, or
  - a **path to an image in your files** (the VFS), e.g.
    `/Pictures/sunset.jpg`.
- **Fit** — how an image wallpaper fills the screen:
  - **cover** — fill the screen, cropping as needed.
  - **contain** — fit the whole image, possibly with letterboxing.
- **Accent color** — the highlight color used across the UI (a hex value like
  `#5b8cff`).

---

## Setting a wallpaper from your files

You can also set a wallpaper directly from the [Files](../apps/files.md) app: open
an image and choose **Set as wallpaper**. That updates this setting for you.

---

## Asking the assistant

Since Appearance is a configuration namespace, the assistant can change it too:

> "Set the wallpaper to /Pictures/forest.jpg and make the accent green."
